# Release Notes: v0.1.0

## Summary

`v0.1.0` is the initial open-source cleanup of the RV32I FPGA contest-oriented project.

## Included

- Initial open-source cleanup.
- Clean repository layout.
- RTL copied without functional changes.
- Vivado reconstruction script added.
- Competition specification notes added.
- Verification plan added.
- Public release checklist added.
- Verification record template added.
- GitHub issue and PR templates added.

## Known Limitations

- No public verification record proving 37/37 pass is included yet.
- No official benchmark or performance number is included.
- No CI workflow is included.
- XCI/IP regeneration still needs validation on a clean machine.
- Memory initialization files are excluded pending authorization review.
- XDC, XCI, testbench, helper modules, and memory materials require human licensing confirmation.

## Licensing Cautions

MIT licensing applies only to files confirmed as original project work. Contest-provided materials, generated Vivado files, memory initialization programs, and third-party code are excluded unless redistribution rights are confirmed.

## Not Claimed

- 37/37 passed.
- Official benchmark result.
- CI passed.
- External adoption.
- Production readiness.
