# v0.1.0-preview Checklist

## Required before tagging

- [ ] `scripts/check_clean_repo.ps1` passes.
- [ ] GitHub Actions Repository Hygiene passes.
- [ ] `git status` is clean.
- [ ] No private memory files are tracked.
- [ ] No Vivado generated artifacts are tracked.
- [ ] Public smoke verification record exists.
- [ ] Public branch-directed verification record exists.
- [ ] Public load/store-directed verification record exists.
- [ ] XDC caveat is documented.
- [ ] XCI caveat is documented.
- [ ] Performance benchmark is not claimed.

## Tag command

```powershell
git tag -a v0.1.0-preview -m "v0.1.0-preview: public RV32I FPGA core preview"
git push origin v0.1.0-preview
```

## GitHub release

Create a GitHub release from tag `v0.1.0-preview`.

Release type: Pre-release.

Do not mark as latest stable release.
