# Security Policy

This is an educational FPGA/RTL repository. Security reports are still welcome when they involve:

- Accidental exposure of private paths, user names, machine names, or credentials.
- Unsafe scripts.
- Redistribution of files with unclear rights.
- Build artifacts that should not be public.

Please open a private report if the hosting platform supports it. Otherwise, open an issue with sensitive details removed and state that private follow-up is needed.
