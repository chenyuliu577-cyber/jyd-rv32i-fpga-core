# Generated Load/Store Directed Memory

This directory contains public, self-generated COE memory images for the
load/store directed test.

Files:

- `irom.coe`
- `dram.coe`

Regenerate them with:

```powershell
python tools/gen_load_store_directed_memory.py
```

These files are repository-owned public test memory images. They are not
private contest memory images and they do not claim complete RV32I 37/37
verification.
